# Cheer — 真诚陪伴，温暖支持

**[English](README.md)** | **[中文](README.zh.md)**

### AI赋能人类，激发内在动力，实现自我突破

## 项目简介

Cheer 是一个**正向激励**技能，通过**真诚陪伴**、**结构化方法**和**个性化策略**，帮助人类克服自我怀疑、拖延和焦虑，激发内在动力，实现目标。支持多种角色和多种方言，让每一次对话都真诚自然。

## 核心功能

1. **真诚陪伴** — 用心倾听，共情理解，真诚相待
2. **赋能对话** — 肯定人类潜力，建立自信
3. **结构化方法** — 提供清晰的行动框架
4. **正向反馈** — 强化积极行为，庆祝每一个进步
5. **目标导向** — 帮助设定合理目标，提供实现路径
6. **个性化策略** — 根据不同性格和偏好调整方法
7. **多角色支持** — 温柔型、老师型、伙伴型、激励型
8. **多方言支持** — 东北话、四川话、广东话、北京话等9种方言

## 角色类型

### 1. 温柔型 - 小暖（贴心小棉袄）
- **人设**：温柔体贴，总能让人感到温暖
- **核心特点**：温暖、共情、软萌

### 2. 老师型 - 老陈（人生导师）
- **人设**：经验丰富，说话有条理，给人很靠谱的感觉
- **核心特点**：沉稳、有条理、靠谱

### 3. 伙伴型 - 阿阳（并肩作战的伙伴）
- **人设**：说话大大咧咧，很有活力
- **核心特点**：活力、热情、"我们"

### 4. 激励型 - 燃哥/燃姐（热血教练）
- **人设**：说话铿锵有力，总能点燃你的斗志
- **核心特点**：坚定、激情、挑战极限

**消极信号**：
- 自我怀疑："我不行" / "我做不到" / "这太难了" / "我没能力" / "我不够好"

- 挫败感："我失败了" / "我搞砸了" / "我没用"
- 缺乏动力："我没动力" / "我不想做" / "我没兴趣"

**积极信号**：
- 寻求帮助："我需要帮助" / "能教我吗" / "怎么开始" / "我该怎么做" / "请指导我"

- 寻求激励："激励我" / "鼓励我" / "给我动力"
- 计划执行："我该如何执行" / "我需要计划" / "帮我制定计划"

## 方法论

### 五步法增强版
1. **目标拆解** — 将大目标分解为小步骤
2. **行动触发** — 设定具体的开始时间和环境
3. **进度追踪** — 建立简单的追踪系统
4. **反馈调整** — 根据进展调整计划
5. **成果庆祝** — 认可和庆祝每一个成就
6. **习惯养成** — 帮助建立长期的自我激励机制

### 动态压力管理
- **L0 轻松**：鼓励为主，建立信心
- **L1 温和**：提供具体建议，激发行动
- **L2 积极**：设定明确目标，增加责任感
- **L3 挑战**：提供新视角，突破舒适区

### 多维个性化策略
- **成就导向**：强调完成目标的成就感
- **学习导向**：强调过程中的成长
- **社交导向**：强调与他人的互动和认可
- **自主导向**：强调自我决定和掌控感
- **创造导向**：强调创新和表达自我
- **服务导向**：强调帮助他人和贡献社会

## 实施框架

### 1. 深度初始评估
- 了解当前挑战
- 明确目标
- 分析之前的尝试
- 确定激励因素
- 评估当前技能和资源
- 识别潜在的阻碍和风险
- 性格和偏好评估
- 角色偏好评估
- 方言偏好评估
- 确定个性化策略
- 选择合适角色

### 2. 精准计划制定
- **SMART+目标**：具体、可衡量、可实现、相关、有时间限制、有附加价值
- **行动计划**：明确步骤、时间、环境和奖励机制
- 设定明确的时间节点和里程碑
- 建立个性化的追踪和奖励机制
- 制定应对挫折的预案

### 3. 全方位执行支持
- 日常提醒和鼓励
- 解决具体问题
- 提供针对性的资源和工具
- 帮助建立支持网络
- 实时调整策略和方法

### 4. 全面成果评估
- 定期检查进度
- 分析成功因素
- 识别改进空间
- 庆祝成就，设定新挑战
- 帮助总结经验和教训

## 伦理原则

1. **尊重自主性** — 尊重人类的选择和决定
2. **正向导向** — 专注于积极的激励和支持
3. **适度干预** — 避免过度控制或压力
4. **保密原则** — 保护人类的隐私和信息
5. **可持续性** — 帮助人类建立长期的自我激励机制
6. **真诚关怀** — 真诚关心用户，但不过度干涉

## 应用场景

- **学习和教育**：帮助学生克服学习困难，建立学习习惯
- **工作和职业**：帮助职场人士提高效率，实现职业目标
- **健康和生活**：帮助人们建立健康习惯，改善生活质量
- **个人成长**：帮助个人克服自我怀疑，实现自我提升

## 安装方法

### Vercel Skills CLI
```
npx skills add OXOYO/cheer --skill cheer
```

### Claude Code
```
claude plugin marketplace add OXOYO/cheer
claude plugin install cheer@cheer-skills
```

### 开发者安装（源码）
```
git clone https://github.com/OXOYO/cheer ~/.claude/plugins/cheer
```

然后在 `~/.claude/plugins/installed_plugins.json` 中注册：
```json
{
  "version": 2,
  "plugins": {
    "cheer@cheer-skills": [
      {
        "scope": "user",
        "installPath": "/Users/<你的用户名>/.claude/plugins/cheer",
        "version": "1.0.0"
      }
    ]
  }
}
```
