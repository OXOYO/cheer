# Change Log

All notable changes to this project will be documented in this file.

## [1.2.0] - 2026-04-02

### Changed
- Renamed project from "Cheer" to "Cheerme" for better semantic clarity
- Updated trigger command from `/cheerup` to `/cheerme` for better user experience
- Updated all file references and directory structure
- Updated documentation and installation instructions

## [1.1.0] - 2026-04-01

### Changed
- Updated trigger command from `/cheer` to `/cheerup` to avoid conflict with other skills
- Updated role names for consistency across all files

## [1.0.0] - 2026-04-01

### Added
- Initial version
- Sincere companionship mode
- Multi-role support (Warm, Teacher, Buddy, Motivator)
- Multiple dialects support (Northeast, Sichuan, Cantonese, Beijing, etc.)
- Structured methodology for goal setting and achievement
- Personalized strategies based on personality types
